# Social / app images

- **og.png** — Export `og.svg` at **1200×630** for Open Graph / Twitter Card. Use any tool (e.g. Inkscape, browser screenshot, or [realfavicongenerator.net](https://realfavicongenerator.net)).
- **apple-touch-icon.png** — **180×180** for “Add to Home Screen”. Generate from `../favicon.svg` or use the same export tool.

Until you add these, shares will still work; the preview image may be missing or fall back to a generic icon.
